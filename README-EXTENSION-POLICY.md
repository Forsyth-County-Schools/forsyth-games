# Forsyth Games - Extension Policy

## Allowed Extensions
Only the following extensions are permitted for use with Forsyth Games:

- **Educational Tools**: Extensions that enhance learning and productivity
- **Accessibility Tools**: Screen readers, color contrast adjusters, etc.
- **Developer Tools**: For educational purposes and debugging

## Blocked Extensions
The following types of extensions are **NOT** permitted:

- **Ad Blockers**: Extensions that block ads or modify the site's appearance
- **Content Modifiers**: Extensions that change game content or functionality
- **Data Harvesters**: Extensions that collect user data or browsing history
- **Gaming Cheats**: Extensions that provide unfair advantages in games
- **Social Media Overlays**: Extensions that add social media features

## Enforcement
- Extension policies are enforced through browser management systems
- Violations may result in restricted access to the gaming platform
- Students should only use approved extensions for educational purposes

## Requesting an Exception
To request an exception for a specific extension:
1. Contact your teacher or IT administrator
2. Provide educational justification for the extension
3. Submit extension details for review
4. Wait for approval from the IT department

## Security Notes
- Extensions from unknown sources are automatically blocked
- All extensions are scanned for security compliance
- Policy updates are applied automatically

For questions about extension policies, please contact the IT department.
